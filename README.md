# GodotVoxelEditor
## 1. Описание проекта
GodotVoxelEditor — это воксельный графический редактор, разработанный на Godot Engine.
## 2. Основные классы проекта
### ComputeHandler
Отвечает за работу с вычислительными шейдерами.
### PaletteHandler
Отвечает за работу с цветовой палитрой.
### UserHandler
Отвечает за обработку пользовательского ввода.
### VoxFormat
Отвечает за работу с форматом vox.
### VoxelGrid
Основной класс, отвечающий за работу с воксельной сеткой.
### VoxelRaytracer
Внутренний класс, реализующий трассировку луча по воксельной сетке.
## 3. Лицензия
Проект распространяется под лицензией **MIT**.
## 4. Инструкция по сборке проекта
1. Загрузить файлы проекта в новую папку
2. Открыть проект, используя движок Godot версии не менее 4.5
3. Установить шаблоны экспорта (Editor -> Manage export templates...)
4. Произвести экспорт на нужную платформу (Project -> Export...) (В проект включен пресет для ОС Windows. Для других систем нужно создать свой пресет)
